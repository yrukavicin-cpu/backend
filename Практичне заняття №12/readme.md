Вирішив взяти SFML (кросплатформна графічна бібліотека)
Вона працює на:
macOS
Windows
Linux

з її допомогою я спробую зробити аналог

інструкція для білду (shell macOS):

brew install sfml@2
clang++ task1.cpp -std=c++17 \
-I/opt/homebrew/opt/sfml@2/include \
-L/opt/homebrew/opt/sfml@2/lib \
-lsfml-graphics -lsfml-window -lsfml-system \
-o task1

./task1

1 — астроїда
2 — фігури
3 — стилі ліній
4 — синусоїди
Esc — вихід
