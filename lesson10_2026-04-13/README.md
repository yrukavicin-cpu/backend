# Lesson 10, 2026-04-13

## Тема
Система введення-виведення в C++.

## Відповіді
1. **Які основні положення системи введення-виведення C++?**  
   У C++ введення і виведення побудовані на основі потоків. Дані передаються через спеціальні об'єкти, наприклад `cin`, `cout`, `cerr`, `clog`.

2. **Що таке потік?**  
   Потік — це послідовність даних, яка передається від джерела до програми або від програми до приймача.

3. **Які потоки відкриваються під час запуску програми мовою C++? Яке їх призначення?**  
   Під час запуску програми стандартно доступні:
   - `cin` — стандартне введення;
   - `cout` — стандартне виведення;
   - `cerr` — виведення повідомлень про помилки;
   - `clog` — виведення службових повідомлень і журналювання.

4. **Які прапори формату ви знаєте? Яке їх призначення? Наведіть приклади.**  
   Прапори формату задають спосіб відображення даних. Наприклад:
   - `ios::dec` — десятковий формат;
   - `ios::hex` — шістнадцятковий формат;
   - `ios::oct` — вісімковий формат;
   - `ios::fixed` — фіксований формат для дробових чисел;
   - `ios::scientific` — науковий формат;
   - `ios::left`, `ios::right` — вирівнювання.

   Приклад:
   ```cpp
   std::cout << std::hex << 255;
   ```

5. **Які функції, що визначають параметри формату ви знаєте? Яке їх призначення? Наведіть приклади застосування.**  
   Основні функції і маніпулятори:
   - `width()` — задає ширину поля;
   - `precision()` — задає точність;
   - `fill()` — задає символ заповнення;
   - `setf()` — встановлює прапори формату;
   - `unsetf()` — скидає прапори;
   - `setw()`, `setprecision()`, `setfill()` — зручні маніпулятори.

   Приклад:
   ```cpp
   std::cout << std::fixed << std::setprecision(2) << 3.14159;
   ```

6. **Які особливості файлового введення-виведення в C++?**  
   Для роботи з файлами використовують класи `ifstream`, `ofstream`, `fstream`. Файл треба відкрити, виконати читання або запис, а потім закрити.

7. **Які функції виконують оператори вставки та вилучення?**  
   Оператор вставки `<<` виводить дані в потік.  
   Оператор вилучення `>>` зчитує дані з потоку.

8. **Як і для чого створюються функції вставки та вилучення?**  
   Такі функції перевантажують для власних класів, щоб можна було зручно вводити і виводити об'єкти через `cin` і `cout`.

9. **Яка загальна форма функції вставки?**  
   Загальний вигляд:

   ```cpp
   std::ostream& operator<<(std::ostream& out, const MyClass& obj);
   ```

10. **Яка загальна форма функції вилучення?**  
    Загальний вигляд:

    ```cpp
    std::istream& operator>>(std::istream& in, MyClass& obj);
    ```

11. **Чи можуть функції вставки та вилучення належати класу, для якого їх створюють?**  
    Зазвичай ні, тому що лівим операндом є об'єкт потоку `ostream` або `istream`. Тому їх частіше роблять зовнішніми функціями, іноді `friend`.

12. **Які типи параметрів та який тип значення, що повертається, у функції вставки та вилучення?**  
    У функції вставки передається посилання на `ostream` і об'єкт класу.  
    У функції вилучення передається посилання на `istream` і об'єкт класу.  
    Обидві функції повертають посилання на потік.

13. **Яка мета створення власних маніпуляторів? Наведіть їх загальну форму.**  
    Власні маніпулятори створюють для зручного форматування виведення або зміни стану потоку.

    Загальна форма:
    ```cpp
    std::ostream& myManip(std::ostream& out) {
        // дії з потоком
        return out;
    }
    ```

14. **Наведіть декілька переваг використання функцій введення-виведення C++ порівняно із системою введення-виведення мови C.**  
    Переваги такі:
    - підтримка перевантаження операторів;
    - зручна робота з класами і об'єктами;
    - краща типобезпечність;
    - простіше форматування;
    - єдина модель роботи для консолі і файлів.
