# Lesson 3, 2026-04-13

## Тема
Самостійне завдання на використання функцій у C++.

## Умова задачі
Завдання (!)

Із урахуванням завдань, сформованих в рамках Вашої курсової роботи наведіть реалізацію прикладів:

- Піднесення числа n в степінь p — це множення числа n на себе p разів. Напишіть функцію з іменем power(), яка в ролі аргументів набуває значення типу double для n та значення типу int для p, повертаючи значення типу double. Для аргументу, що відповідає степеню числа, задайте значення за умовчанням, рівне 2, щоб за відсутності показника степеню за умови виклику функції число n підносилось в квадрат. Напишіть функцію main(), яка здійснює запит у користувача введення аргументів для функції power(), і відобразіть на екрані результати її роботи.
- Напишіть функцію з іменем zeroSmaller(), в яку передаються за допомогою посилань два аргументи типу int, яка присвоює меншому з аргументів нульове значення. Напишіть програму для перевірки роботи функції.
- Напишіть функцію, що набуває в якості аргументів два значення типу Distance та повертає значення найбільшого з аргументів.

Результати роботи внесіть в робочу таблицю (див. прикріплення нижче)

## Компіляція і запуск
```bash
clang++ -std=c++17 main.cpp -o lesson3
./lesson3
```

## Висновок
У роботі реалізовано три приклади: функцію піднесення до степеня зі значенням параметра за умовчанням, функцію зміни меншого аргументу через посилання та функцію порівняння двох значень типу `Distance`.
