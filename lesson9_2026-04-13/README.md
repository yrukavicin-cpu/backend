# Lesson 9, 2026-04-13

## Тема
Оброблення виняткових ситуацій.

## Відповіді
1. **Що таке виняткова ситуація?**  
   Виняткова ситуація — це помилка або нестандартна подія, яка виникає під час виконання програми і заважає її нормальній роботі.

2. **Запишіть загальні форми операторів `try`, `catch` і `throw`. Опишіть їх функції.**  
   Загальний вигляд:

   ```cpp
   try {
       // код, де може виникнути помилка
   }
   catch (тип_винятку змінна) {
       // обробка винятку
   }
   ```

   ```cpp
   throw вираз;
   ```

   `try` містить код, де може виникнути виняток.  
   `catch` перехоплює і обробляє виняток.  
   `throw` генерує виняток.

3. **Як усічна робота операторів `try`, `catch` і `throw` забезпечує в C++ оброблення виняткових ситуацій?**  
   Код, у якому може статися помилка, поміщають у блок `try`. Якщо виникає виняток, виконується `throw`, і керування передається у відповідний блок `catch`, де ця ситуація обробляється.

4. **Чи можна використовувати `throw`, якщо виконання програми не проходить через блок `try`?**  
   Так, можна, але якщо виняток ніде не буде перехоплений, програма аварійно завершиться.

5. **Яка форма оператора `catch` буде обробляти всі типи виняткових ситуацій?**  
   Для перехоплення будь-яких винятків використовується:

   ```cpp
   catch (...) {
       // обробка всіх винятків
   }
   ```

6. **Яким буде результат генерації виняткової ситуації, для якої не задано відповідного оператора `catch`?**  
   Якщо відповідного `catch` немає, виняток вважається необробленим, і програма завершується з помилкою.
