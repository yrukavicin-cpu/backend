# Lesson 1, 2026-04-13

## Тема
Самостійне завдання на використання STL `list`.

## Умова задачі
Задано натуральне число `n` і дійсні числа `a1, a2, ..., an`.

Потрібно:

- якщо у послідовності є хоча б один елемент, менший за `-3`, то всі від'ємні елементи замінити їх квадратами;
- інакше збільшити всі елементи на `0.1`;
- результат вивести у зворотному порядку.

## Реалізація
У програмі використано контейнер `std::list<double>`.

Алгоритм роботи:

1. Зчитати кількість елементів `n`.
2. Заповнити двонаправлений список числами.
3. Перевірити, чи існує хоча б один елемент `< -3`.
4. Якщо такий елемент є, змінити тільки від'ємні числа на їх квадрати.
5. Якщо такого елемента немає, додати `0.1` до кожного елемента.
6. Вивести список у зворотному порядку.

## Файли
- `main.cpp` — розв'язання задачі мовою C++.

## Компіляція і запуск

```bash
clang++ -std=c++17 main.cpp -o lesson1
```

Вхід:
```text

```

```text

```

## Приклад 2

```text
4

```

```text
3.10 -2.40 0.10 1.30
```
