# Практичне заняття №4, 2026-04-13

## Тема
Шаблони. Бібліотека стандартних шаблонів.

## Мета роботи
Ознайомитися з контейнерами STL, ітераторами, алгоритмами та базовими шаблонними класами стандартної бібліотеки C++.

## Відповіді на запитання
1. **Які елементи входять до складу бібліотеки стандартних шаблонів?**  
   До STL входять контейнери, ітератори, алгоритми, алокатори, функціональні об'єкти, предикати та адаптери.

2. **Дати характеристику об’єктам контейнер, ітератор, алгоритм. Типи ітераторів.**  
   Контейнер зберігає набір елементів. Ітератор дає змогу послідовно переглядати елементи контейнера. Алгоритм виконує дії над елементами контейнера, наприклад пошук або сортування. Ітератори бувають введення, виведення, однонапрямлені, двонапрямлені, довільного доступу та зворотні.

3. **Дати характеристику об’єктам алокатор, функтор, предикат, адаптер.**  
   Алокатор відповідає за роботу з пам'яттю. Функтор — це об'єкт, який можна викликати як функцію. Предикат — це функція або функтор, що повертає логічне значення. Адаптер змінює інтерфейс або поведінку іншого елемента STL.

4. **Шаблонний клас `string` і його методи для роботи з рядками символів.**  
   `string` використовується для роботи з рядками змінної довжини. Основні методи: `size()`, `length()`, `insert()`, `erase()`, `substr()`, `find()`, `append()`, `clear()`.

5. **В чому перевага і недоліки використання об’єктів типу `string`.**  
   Переваги: зручність, автоматичне керування пам'яттю, багато готових методів. Недоліки: іноді більші витрати пам'яті і менший контроль, ніж при роботі з масивом `char`.

6. **Шаблонний клас `vector` і його методи для роботи з динамічними векторами.**  
   `vector` — це динамічний масив. Основні методи: `push_back()`, `size()`, `resize()`, `insert()`, `erase()`, `clear()`, `front()`, `back()`.

7. **Призначення алгоритмів для роботи з колекціями.**  
   Алгоритми потрібні для обробки колекцій: пошуку, сортування, копіювання, заміни, підрахунку елементів.

8. **Масив значень і його методи.**  
   Масив значень `valarray` використовується для числових обчислень над наборами даних. Основні методи: `size()`, `resize()`, `min()`, `max()`, `sum()`.

9. **Стек і його методи.**  
   Стек працює за принципом LIFO. Основні методи: `push()`, `pop()`, `top()`, `size()`, `empty()`.

10. **Список і його методи.**  
    `list` — це двонаправлений список. Основні методи: `push_front()`, `push_back()`, `pop_front()`, `pop_back()`, `insert()`, `erase()`, `sort()`, `clear()`.

11. **Черги і деки та їх методи.**  
    Черга працює за принципом FIFO. Основні методи черги: `push()`, `pop()`, `front()`, `back()`. Дек дозволяє додавати і вилучати елементи з обох кінців. Основні методи деку: `push_front()`, `push_back()`, `pop_front()`, `pop_back()`.

12. **Асоціативні відображення.**  
    Асоціативні відображення зберігають дані у форматі "ключ-значення". Для цього використовується контейнер `map`.

13. **Множини і множини бітів.**  
    `set` зберігає унікальні елементи у відсортованому вигляді. `bitset` використовується для роботи з набором бітів фіксованої довжини.
